# Campus CRM Demo

This is a small CRM-themed web project built with HTML, CSS, and JavaScript.

The project has three main files:

- `index.html` contains the page structure.
- `style.css` controls the visual design.
- `script.js` stores contacts, renders the list, adds new contacts, and filters by status.

## How to Run

Open `index.html` in a browser.

No install step is needed.

## Possible Improvements

- Add a search box for contact names.
- Add a delete button for each contact.
- Add a priority field such as Low, Medium, and High.
- Count how many contacts are in each status.
- Save contacts in `localStorage` so they remain after refresh.
